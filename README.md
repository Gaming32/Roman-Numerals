# Installing
``` shell
$ python3 -m pip install Roman-Numerals-Simple
```

# Using
``` python
>>> import roman
>>> roman.int_to_roman(947)
'CMXLVII'
>>> roman.roman_to_int('CMXLVII')
947
```

# Command Line
``` shell
$ python3 -m roman
5377
MMMMMCCCLXXVII
5377 True
$ python3 -m roman 947
947
CMXLVII
947 True
```
